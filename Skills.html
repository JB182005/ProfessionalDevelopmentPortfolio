Anime Street Fighter Project & 2D Shooter Game Project

Street Fighter Game

2D Shooter Game

For my major dev projects, I built two games from scratch using Python and Pygame—one’s a 2-player anime-style fighting game, and the other’s a 2D side-scrolling shooter packed with action, AI, and explosions. Both games showcase how far I’ve come with game development and my ambition to enter the industry.

Anime Street Fighter Game

Inspired by classics like Street Fighter and Jump Force, this 2-player fighting game lets users pick between Naruto and Luffy in a head-to-head battle. I built character classes, implemented collision detection, health bars, animated sprite sheets, special moves, and more. The game starts with a clean main menu and features a pause screen with video/audio settings and key binding options.

  • Character classes with health, movement, and special moves
  • Animated sprite sheets with punch, kick, block, and jump mechanics
  • Sound FX for attacks and background music for immersion
  • Score tracking and a round-based win system
  • Custom menus and pause functionality with options

2D Shooter Game

This game is all about action. You're a soldier dropped into enemy territory with limited ammo and grenades. Enemies shoot on sight, there's water that'll kill you, and items scattered around to boost your health and supplies. It includes multiple levels built from CSV files and even a custom level editor.

  • Tile-map based world with platforming, AI enemies, and environmental hazards
  • Player can shoot, throw grenades, and collect pickups
  • Multiple levels with increasing difficulty and a clear exit goal
  • Sound FX for everything gunfire, jumping, dying and background music
  • Custom level editor where users can make and save their own maps

Both games pushed my coding ability, design skills, and creativity. They’ve been a launchpad for my future in game dev. Whether it’s enemy AI, responsive menus, or sprite animations.
